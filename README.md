# 🚛 DriveDeal

This project is a custom multilingual **WordPress** website developed for **DriveDeal.lv**, a company offering **waste removal** and **transportation** services in Latvia.

The website was built using **WordPress**, **Astra theme** template, **WooCommerce** and **Elementor**, with the visual design based on a **Figma layout** created by my colleague. 
The website is available in **3 languages**: Latvian, Russian and English.

Its main goal is to make service ordering simple and convenient: users can calculate the cost of waste removal online, submit an order, choose a preferred date and time, and pay online. For transportation services, the site also includes a dedicated request form.

## ✨ Main Features

### 🧮 Waste Removal Price Calculator
Users can order waste removal through an interactive flow with automatic price calculation.

**Functionality includes:**
- selecting waste type and items
- choosing quantity, address, delivery to floor
- automatic price calculation
- choosing date and time
- online payment option
- confirmation by email with attached invoice

---

### 🚚 Transportation Request Form
The website also includes a separate transportation service request form.

Users can:
- enter pickup and delivery locations
- provide contact details
- add comments about cargo, weight, quantity, or timing
- submit a transportation request for follow-up
